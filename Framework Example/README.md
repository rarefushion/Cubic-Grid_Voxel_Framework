Program.cs breaks down how to use the Framework, Renderer, FileManagement and Debugging modules using Silk.NET to create a functional cubic voxel world the generates around the camera.

### Controls
Escape to close.
E to unlock cursor.
F3 to show debug menu.
WASD to move forward, back, left and right.
Shift to move down.
Space to move up.
Scroll wheel to increase or decrease speed.
